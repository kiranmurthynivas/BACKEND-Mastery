# REST Posts App

A beginner-friendly REST CRUD project using Express, EJS, UUID, and method-override.

## Topics Practiced

- REST
- CRUD operations
- Index route
- New route
- Create route
- Show route
- Edit route
- Update route
- Destroy route
- Redirect
- UUID
- method-override

## Routes

| Method | Route | Description |
|---|---|---|
| GET | /posts | Show all posts |
| GET | /posts/new | Form to create post |
| POST | /posts | Create new post |
| GET | /posts/:id | Show one post |
| GET | /posts/:id/edit | Form to edit post |
| PATCH | /posts/:id | Update post |
| DELETE | /posts/:id | Delete post |
