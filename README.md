# BACKEND-Mastery

This repository documents my backend learning journey from Node.js basics to MongoDB with Express CRUD applications.

I followed a practical project-based approach: learn a concept, build a small project, commit the work, and revise with notes.

## Goal

To build strong backend fundamentals for MERN stack development and become job-ready by learning:

- Node.js
- Express.js
- EJS
- REST APIs
- SQL
- MySQL with Node.js
- MongoDB
- Mongoose
- MongoDB with Express
